# To-Do List in C++ using Linked Lists

A console-based To-Do List application built in C++ as part of my Data Structures and Algorithms coursework. This project uses a singly linked list to manage tasks with different priority levels.

## Features
- Add tasks with priority levels (High / Medium / Low)
- View all tasks along with their status
- Mark tasks as completed
- Delete tasks
- Save tasks to a file
- Load tasks automatically from a file

## Data Structure Used
**Singly Linked List** — each task is represented as a node containing:
- Task name
- Priority level
- Completion status
- Pointer to the next node

## Concepts Learned
- Implementation of a singly linked list from scratch
- Dynamic memory allocation using pointers
- Insertion and deletion operations in linked lists
- File handling in C++ (saving and loading data)
- Building menu-driven console applications

## How to Run
1. Clone this repository or download the source code.
2. Open the project in Code::Blocks or any C++ compiler.
3. Compile and run the program.
